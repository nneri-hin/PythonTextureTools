# PythonTextureTools

**概要**

ハイトマップからノーマルマップ変換するだけのプログラムです
一枚読み込んで一枚出力するだけです

**必要環境**

Python3系列
OpenCV
numpy

**使用方法**

```
python craete_normalmap.py `input.png` `output.png`
```
